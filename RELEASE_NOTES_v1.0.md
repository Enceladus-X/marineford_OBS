# Marineford OBS v1.0

첫 현장 투입용 릴리스입니다.

## 포함 기능

- 송출용 패널에서 대회명, 플레이어 이름, 이미지, 점수, 타이머를 관리
- 송출용 패널 안에서 태블릿 연결 QR 표시
- 태블릿 패널에서 플레이어 닉네임/덱 입력, 듀얼 시작, 승패 보고, 다음 라운드 준비 처리
- OBS 브라우저 소스용 1920x1080 오버레이 제공
- 태블릿 이벤트 기반 편집 보조 로그와 CSV/챕터/ffmpeg 초안 export 제공
- Windows용 단일 실행 파일 `Marineford_OBS_v1.0.exe` 제공

## 현장 실행법

1. `Marineford_OBS_v1.0.exe`를 다운로드합니다.
2. 송출컴에서 EXE를 더블클릭합니다.
3. 자동으로 열리는 송출용 패널에서 태블릿 연결 QR을 스캔합니다.
4. OBS 브라우저 소스 URL을 `http://127.0.0.1:8000/overlay.html`로 설정합니다.

## 주의사항

- 송출컴과 태블릿은 같은 Wi-Fi 또는 같은 공유기에 연결되어 있어야 합니다.
- Windows 방화벽 알림이 뜨면 실행 파일의 개인 네트워크 접근을 허용해야 합니다.
- 실행 중 EXE 옆에 생성되는 `state.json`, `events.jsonl`, `images/`는 현장 상태 저장용 파일입니다.
