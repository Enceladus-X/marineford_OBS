# Marineford OBS v1.1

현장 연결성과 라운드 운영 흐름을 보강한 릴리스입니다.

## 변경 사항

- QR 코드를 검증된 QR 생성 라이브러리 기반 SVG로 교체해 태블릿 카메라 인식률 개선
- 태블릿 세트 스코어 화면 상단에 듀얼 타이머 표시
- 송출용 패널에 덱 이름 입력과 덱 이미지 프리셋 저장 기능 추가
- 저장된 덱 프리셋을 송출용 패널과 태블릿 덱 입력 자동완성에 사용
- 태블릿에서 저장된 덱 이름을 입력하면 해당 덱의 이미지 2장을 자동 적용
- OBS 오버레이의 사이드 교체중 화면을 전체 암전 전환 화면으로 변경
- 다음 라운드 준비 상태에서 OBS 오버레이에 `다음 라운드 대기중` 암전 화면 표시

## 현장 실행법

1. `Marineford_OBS_v1.1.exe`를 다운로드합니다.
2. 송출컴에서 EXE를 더블클릭합니다.
3. 자동으로 열리는 송출용 패널에서 태블릿 연결 QR을 스캔합니다.
4. OBS 브라우저 소스 URL을 `http://127.0.0.1:8000/overlay.html`로 설정합니다.

## 주의사항

- 송출컴과 태블릿은 같은 Wi-Fi 또는 같은 공유기에 연결되어 있어야 합니다.
- Windows 방화벽 알림이 뜨면 실행 파일의 개인 네트워크 접근을 허용해야 합니다.
- 실행 중 EXE 옆에 생성되는 `state.json`, `events.jsonl`, `deck_presets.json`, `images/`는 현장 상태 저장용 파일입니다.
