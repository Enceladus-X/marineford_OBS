# Marineford OBS v1.2

현장 태블릿 운용과 OBS 연결 가시성을 보강한 릴리스입니다.

## 변경 사항

- 송출 패널의 태블릿 QR을 인라인 SVG 렌더링 방식으로 보강해 QR 표시 안정성 개선
- 송출 패널 상단에 OBS 브라우저 소스 URL, 복사, 미리보기 기능 추가
- 태블릿 우측 상단에 고급 패널 추가
- 고급 패널에 전체화면 전환, 마지막 승패 보고 취소, 수동 점수 조절, 플레이어 정보 수정 기능 배치
- 태블릿에 저지 호출 버튼 추가
- 저지 호출 시 타이머를 멈추고 OBS에 `저지 호출 중` 암전 화면 표시
- 저지 호출 종료 시 호출 전 상태와 타이머 진행 여부로 복귀
- BO1/BO3, 기본 타이머, 엑스트라 턴 시간 설정 추가
- 게임별 룰 프리셋 저장 및 자동완성 추가
- 타이머 종료 후 엑스트라 턴 시간을 `EX mm:ss` 형식으로 표시
- 태블릿 가로 화면에서 브라우저 상단 표시줄로 하단이 잘리는 문제를 줄이도록 동적 높이와 스크롤/전체화면 대응 추가

## 현장 실행법

1. `Marineford_OBS_v1.2.exe`를 다운로드합니다.
2. 송출컴에서 EXE를 더블클릭합니다.
3. 송출용 패널의 OBS URL을 OBS 브라우저 소스에 입력합니다.
4. 송출용 패널의 QR을 태블릿으로 스캔합니다.

## 주의사항

- 송출컴과 태블릿은 같은 Wi-Fi 또는 같은 공유기에 연결되어 있어야 합니다.
- Windows 방화벽 알림이 뜨면 실행 파일의 개인 네트워크 접근을 허용해야 합니다.
- 실행 중 EXE 옆에 생성되는 `state.json`, `events.jsonl`, `deck_presets.json`, `rule_presets.json`, `images/`는 현장 상태 저장용 파일입니다.
