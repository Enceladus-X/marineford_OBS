# Marineford OBS v1.3

OBS 레이아웃 충돌을 줄이고 덱 이미지 편집 흐름을 보강한 릴리스입니다.

## 변경 사항

- OBS 오버레이에서 닉네임을 더 크게 표시하고 상단으로 이동해 세트 스코어와 겹치지 않도록 조정
- 명함 이미지 영역을 제거하고 해당 영역의 역할을 큰 닉네임 표시로 대체
- 덱 이미지 슬롯을 `cover` 방식으로 꽉 차게 표시하도록 변경
- 송출 패널 덱 이미지 업로드 시 이미지 자르기 패널 추가
- 자르기 패널에서 위치 조정, 확대, 좌우반전 후 이미지 사용 확정 가능
- 이미지 1은 세로형 슬롯 기준으로 자르도록 기본 비율 조정
- 각 덱 이미지 미리보기에 X 버튼을 추가해 입력을 즉시 삭제할 수 있도록 개선
- 다음 듀얼 시작 시 타이머가 기본 타이머로 초기화되도록 서버 액션 수정
- 송출 패널의 깨진 한국어 문구를 정리

## 현장 실행법

1. `Marineford_OBS_v1.3.exe`를 다운로드합니다.
2. 송출컴에서 EXE를 더블클릭합니다.
3. 송출용 패널의 OBS URL을 OBS 브라우저 소스에 입력합니다.
4. 송출용 패널의 QR을 태블릿으로 스캔합니다.

## 주의사항

- 송출컴과 태블릿은 같은 Wi-Fi 또는 같은 공유기에 연결되어 있어야 합니다.
- Windows 방화벽 알림이 뜨면 실행 파일의 개인 네트워크 접근을 허용해야 합니다.
- 실행 중 EXE 옆에 생성되는 `state.json`, `events.jsonl`, `deck_presets.json`, `rule_presets.json`, `images/`는 현장 상태 저장용 파일입니다.
