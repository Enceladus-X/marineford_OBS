# Marineford OBS v1.4

현장 태블릿 조작과 OBS 송출 화면 안정성을 보강한 릴리스입니다.

## 변경 사항

- 저지 호출 중 타이머가 다시 흐르거나 상태가 풀리는 문제를 수정했습니다.
- 매치 종료/강제 종료 후 오래된 패널 갱신이 상태를 되돌리는 문제를 막았습니다.
- 저지 호출 화면과 사이드 교체 화면을 전체 암전 대신 중앙 반투명 띠지로 표시하도록 바꿨습니다.
- 다음 라운드 대기 상태는 전체 암전 화면으로 유지합니다.
- 송출 오버레이의 대회명, 타이머, 듀얼/세트 정보 표시를 크게 조정했습니다.
- 태블릿 고급 패널의 수동 점수 조절을 좌우형 `- 점수 +` 배치로 정리했습니다.
- 태블릿의 플레이어 정보 수정 버튼 문구를 명확하게 바꿨습니다.
- 태블릿의 상단 설명성 문구와 다음 라운드 준비 확인창을 제거했습니다.
- 플레이어 설정 입력창 크기를 줄이고 자동 포커스를 제거해 모바일 키보드가 화면을 덮는 문제를 완화했습니다.
- 덱 이름 입력 시 상대에게 보이지 않는다는 안내와 덱 프리셋 빠른 선택 칩을 추가했습니다.
- 듀얼 시작 전 하단 안내문구를 제거하고 `듀얼 시작` 버튼을 크게 배치했습니다.
- 저장된 덱 이름-이미지 프리셋이 덱 이름만으로 자동 적용되는 흐름을 검증했습니다.

## 사용 방법

1. `Marineford_OBS_v1.4.exe`를 다운로드합니다.
2. 송출컴에서 실행합니다.
3. 방송 컨트롤 패널에서 태블릿 QR을 열고 태블릿으로 접속합니다.
4. OBS 브라우저 소스에는 컨트롤 패널의 OBS 안내 URL을 등록합니다.
