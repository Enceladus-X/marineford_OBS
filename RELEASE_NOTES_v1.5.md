# Marineford OBS v1.5

이미지 편집 안정성과 현장 상태 동기화를 중심으로 다듬은 릴리스입니다.

## 주요 변경 사항

- crop 결과를 다시 원본으로 사용하던 문제를 수정했습니다.
- 업로드 원본과 송출용 crop을 별도 파일로 저장하고, 다시 편집할 때 항상 원본을 불러옵니다.
- crop 창에 실제 송출 영역의 종횡비, 픽셀 크기, 안전선과 3분할 가이드를 표시합니다.
- crop 캔버스 CSS 비율을 출력 비율과 고정해 이미지가 늘어나던 문제를 수정했습니다.
- 컨트롤 패널의 오래된 전체 상태가 태블릿 점수, 승패, 타이머를 되돌리던 동기화 충돌을 수정했습니다.
- 타이머를 서버 기준으로 계산하고 설정 저장은 필요한 항목만 부분 갱신하도록 변경했습니다.
- 승패 보고 후 점수가 `0:0`으로 돌아가거나 타이머가 증감하던 문제를 수정했습니다.
- 저지 호출, 매치 종료, 다음 듀얼 시작 중에도 상태와 타이머가 안정적으로 유지됩니다.
- 태블릿 플레이어 이름을 위로 이동하고 글자 크기를 키웠습니다.
- 송출 화면의 덱 이름 글자 크기를 키웠습니다.
- 컨트롤 패널을 좌우 2열의 압축된 모던 다크 레이아웃으로 개편했습니다.
- 컨트롤 패널 상단에서 송출 화면, 태블릿 화면, 로그 패널을 바로 열 수 있습니다.
- 저장된 덱 프리셋을 검색하고 왼쪽 또는 오른쪽에 명시적으로 불러올 수 있습니다.
- 룰 프리셋 저장 상태와 저장 목록을 표시하고, 선택한 프리셋을 불러오는 버튼을 추가했습니다.
- 편집 로그 패널을 모던 다크 디자인으로 개편하고 패널 간 이동 버튼을 추가했습니다.
- 공개 EXE에 현장 업로드 이미지가 포함되지 않도록 릴리스 빌드를 정리했습니다.

## 설치

1. `Marineford_OBS_v1.5.exe`를 다운로드합니다.
2. 송출컴의 원하는 폴더에서 실행합니다.
3. 자동으로 열린 컨트롤 패널에서 태블릿 QR을 스캔합니다.
4. OBS 브라우저 소스에 컨트롤 패널의 오버레이 URL을 등록합니다.

## 권장 OBS 설정

- URL: `http://127.0.0.1:8000/overlay.html`
- 너비: `1920`
- 높이: `1080`
