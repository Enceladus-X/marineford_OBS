Marineford OBS

TCG 매장 송출 화면 미리보기

실제 송출컴에서는 로컬 프로그램이 OBS와 태블릿을 연결합니다. 이 미리보기는 웹 페이지에 넣기 위한 정적 데모이며, 매장 데이터나 로컬 프리셋을 읽지 않습니다.

Windows 실행 파일 다운로드 릴리스 노트
1920 × 1080 OBS 브라우저 소스 기준

현장 사용 흐름

  1. 송출컴에서 Marineford OBS 실행 파일을 켭니다.
  2. 컨트롤 패널에서 OBS URL과 태블릿 QR을 확인합니다.
  3. OBS에는 오버레이 URL을 브라우저 소스로 넣습니다.
  4. 태블릿에서 듀얼 시작, 승패 보고, 저지 호출을 조작합니다.
