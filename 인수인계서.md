## 1. 시스템 개요

OBS 대회 방송용 실시간 오버레이 시스템입니다.  
Python 로컬 서버를 중간에 두고, 크롬 컨트롤 패널과 OBS 브라우저 소스가 서버를 통해 실시간으로 연동됩니다.

```
[운영자 크롬] → control.html → POST /state → server.py → state.json
                                                               ↓
                              [OBS 브라우저] ← GET /state ← server.py
```

---

## 2. 파일 구성

| 파일 | 역할 |
|---|---|
| `server.py` | Python 로컬 웹서버. 반드시 먼저 실행 |
| `overlay.html` | OBS 브라우저 소스에 등록하는 방송 화면 |
| `control.html` | 운영자가 크롬으로 여는 컨트롤 패널 |
| `state.json` | 현재 방송 상태 저장 파일 (자동 생성) |
| `images/` | 업로드된 이미지 저장 폴더 (자동 생성) |

> 모든 파일은 **같은 폴더** 안에 있어야 합니다.

## 3. 매 방송 시작 전 실행 순서

### 3-1. 서버 실행
명령 프롬프트(cmd)를 열고:

```
cd C:\Users\Holick\Desktop\Files\TASKS\PROJECTS\marineford_OBS
python server.py
```

아래처럼 뜨면 정상:
```
[마린포드] 서버 시작: http://localhost:8000
  오버레이: http://localhost:8000/overlay.html
  컨트롤:   http://localhost:8000/control.html
  종료하려면 Ctrl+C
```

> ⚠️ cmd 창은 방송 중 절대 닫으면 안 됩니다. 닫으면 서버가 꺼집니다.

### 3-2. 컨트롤 패널 열기
크롬 주소창에 입력:
```
http://localhost:8000/control.html
```

### 3-3. OBS 설정 확인
- 브라우저 소스 URL: `http://localhost:8000/overlay.html`
- 너비: `1920` / 높이: `1080`
- 로컬 파일 체크: **해제**

### 3-4. OBS 브라우저 소스 새로 고침
소스 우클릭 → **새로 고침**

## 4. 컨트롤 패널 사용법

### 대회 정보
- **대회명**: 하단 좌측에 표시되는 대회 이름 입력

### 플레이어 정보
- **닉네임**: 상단 좌/우에 크게 표시
- **명함 이미지**: 닉네임 옆에 붙는 캐릭터/일러스트 이미지 (260×100px 슬롯)
  - 클릭하여 파일 선택, 또는 경로/URL 직접 입력

### 사이드 이미지 패널 (좌/우 검정 구역)
- **슬롯 1만 입력** → 전체를 꽉 채우는 단독 이미지 (cover)
- **슬롯 1+2 둘 다 입력** → 상/하 절반 분할 (상단 cover, 하단 contain)

### 스코어
- `+` / `−` 버튼으로 조작
- **스코어 초기화** 버튼으로 0:0 리셋

### 타이머
- 프리셋: 40 / 35 / 30 / 25 / 20분
- 직접 입력 후 설정 버튼
- ▶ 시작 / ⏸ 일시정지 / ↺ 리셋
- 3분 이하: 주황색 경고 / 0 이하: 빨간색 + 점멸

## 6. 이미지 업로드 규칙

- 컨트롤 패널에서 파일 선택 시 자동으로 `images/` 폴더에 업로드
- OBS는 `http://localhost:8000/images/파일명` 으로 접근
- 파일을 직접 `marinford-obs/images/` 폴더에 넣고 경로 입력도 가능
- 외부 URL (https://...) 직접 입력도 지원

## 7. 라운드 교체 루틴

매 라운드마다 이 순서로:

1. 플레이어 닉네임 수정
2. 명함 이미지 교체 (필요 시)
3. 사이드 이미지 교체 (필요 시)
4. **스코어 초기화**
5. **타이머 리셋** 후 시작

## 8. 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| 컨트롤 패널 수정이 OBS에 안 반영됨 | 서버가 꺼져 있음 | cmd에서 `python server.py` 재실행 |
| OBS 화면이 비어있음 | URL이 로컬파일(`file:///`)로 설정됨 | 브라우저 소스 URL을 `http://localhost:8000/overlay.html`로 변경 |
| 이미지가 OBS에서 안 보임 | blob URL 사용 | 컨트롤 패널에서 파일 선택 버튼으로 업로드 (자동 처리) |
| 서버 포트 충돌 오류 | 8000번 포트 이미 사용 중 | 기존 cmd 창 닫고 재실행, 또는 `server.py` 내 port = 8001로 변경 |
| state.json 오류 | 파일 손상 | `state.json` 삭제 후 서버 재시작 (자동 재생성) |
